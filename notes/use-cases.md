# Use Cases

This catalog is a ServiceNow endpoint inventory for API discovery, integration design, automation, governance, reporting, migration planning, and operational support.

## API Discovery

- Understand common ServiceNow endpoint families.
- Identify which table aliases matter for ITSM, CMDB, identity, metadata, audit, and knowledge use cases.
- Distinguish reusable parameterized API families from fixed product APIs.

## Integration And Automation

- Read and write table records.
- Upload and download attachments.
- Submit service catalog requests.
- Run aggregate counts and grouped reports.
- Load data through import set staging tables.

## Reporting And Governance

- Inventory tables, dictionary fields, choices, users, groups, roles, and assignments.
- Validate process data across incident, problem, change, task, request, requested item, and catalog task tables.
- Review audit and journal history where permissions allow it.

## Migration

The same catalog can be filtered for migration workflows:

- Discover schema and reference data.
- Extract source records and attachments.
- Transform users, groups, choices, states, categories, CIs, and references.
- Load through Table API or Import Set API.
- Validate counts and relationships using Table API and Aggregate API.
